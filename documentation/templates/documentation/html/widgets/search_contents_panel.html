Search & Query the PhenoImageShare Repository

PhenoImageSgare presently allows freetext search through its database, while supporting filtering based on eight data types, namely Sample type, Image type, Imaging method, Development stage, Anatomy, Gene and Phenotype.

Users can browse the PhIS repository using the filters or perform a freetext search or combination of both. Freetext could range from a gene or gene symbol, phenotype description or anatomy term of interest to the user. Each top level data type has 1 or more lower-level data fields allowing for further data filtering.


Freetext search - phenotype term search

Entering freetext such as gene or allele symbol, phenotype, anatomy or annotation term in the main input field of the search interface produces and displays an autosuggested list of terms indexed in the database. Figure below shows a freetext search for phenotype term "abnormal hair cycle".


Freetext search & facet controls - gene symbol search & data filtering

Following a freetext search, data obtained in search result can be further filtered using the facets. The figure below shows a freetext search for a gene Stx8 followed by filtering based on Sample/Image types (Mutant/Expression) and the Development stage (postnatal).

Results obtained from the gene search can be filtered by clicking/opening any of the facets for which data exists and ticking any of the subfacets under the facet.

Browsing through the database & faceting

You can browse the database to your desired results set using facets. The facets allow you to drill down to the desired results without having to type in freetexts in the generic search box. Exploring the database could start right from the landing page, where you can click on any point of interest on the landing charts. These charts present the visual summaries of the dataset in PhenoImageShare repository.


Browsing from landing page

You can click on any point on the charts and this would take you to the search page where the corresponding records are displayed based on applied filters.

Browsing using facets (checkboxes & freetext inputs)

Data in the PhIS repository can be browsed and/or filtered using the facets and the subfacets without having to type in freetext in generic search box. Facets and subfacets allows you to navigate to the records of interest by checking the facet boxes or making a facet-based freetext search (e.g for Gene, Anatomy, or Phenotype using facets equivalents). Clicking on the top-level facets (such as Mutants, Wildtype, Imaging method, or Development stage) will open up subfilters (such as Expression or Phenotype subfacets of Mutants / Wiltype main facets) for filtering. Checking or ticking a subfacet of interest will return and display a narrowed down result set based on selected field values.
