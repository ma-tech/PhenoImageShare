Overview

PhenoImageSgare is a phenotype image annotation, sharing and discovery platform. It provides tools for querying the images, and image metadata, stored within its repository.

PhenoImageShare also provides a submission tool to allow users to submit new images and their associated annotaitons. Registration and Login is required before a user can submit new annotations or images.

Requirements

PhenoImageShare requires the following:

  • Python (2.6.5+ or 2.7)
  • Django (1.4.2+, 1.5, 1.6, 1.7)
  • Apache (2.2.15+)
  • mod_wsgi (3.2+)
  • Python virtualenv (1.10.1+)
  • Simplejson (2.0.9+)
  • Python urllib2 (0.1.6+)
  • pip (1.3.1+)

The following packages are optional:

  • Markdown (2.1.0+) - Markdown support for the browsable API.
  • Pypandoc (0.8.2+) - Multi-format documentation support.
  • PyMongo (2.7.0+) - Python support for MongDB.

Note: The PyMongo Python package is required only for spatial annotations.

Installation

PhenoImageShare UI can be installed by the following steps.

Python Packages

Install using pip, including any optional packages you want...

pip install django==1.6 # Django 1.6 or above.
pip install simplejson==2.0.9       # JSON library.
pip install python-urllib2==0.1.6  # URL library.

...or clone the project from github.

git clone https://github.com/ma-tech/PhenoImageShare.git

Add the following to your INSTALLED_APPS variable in your settings.py file:

  • 'queries'
  • 'annotation'
  • 'documentation'
  • 'submission'
INSTALLED_APPS = (
      ...
      'queries',
      'annotation',
      'documentation',
      'submission',
)
